# DSS - Sistema de Gestão de Restaurante

Projeto desenvolvido no âmbito da unidade curricular de Desenvolvimento de Sistemas de Software da Licenciatura em Engenharia Informática da Universidade do Minho.

## Descrição

Sistema de gestão de restaurante, desenvolvido com o objetivo de apoiar processos como gestão de menus, pedidos, reservas, funcionários, clientes e operações internas do restaurante.

## Funcionalidades

- Gestão de clientes
- Gestão de funcionários
- Gestão de mesas/reservas
- Gestão de pedidos
- Gestão de menus/produtos
- Apoio às operações administrativas do restaurante

## Tecnologias utilizadas

- Java
- MySQL/MariaDB
- Maven/Gradle
- Git/GitHub

## Como executar

1. Clonar o repositório.
2. Configurar a base de dados.
3. Atualizar as credenciais de ligação.
4. Executar a aplicação.
